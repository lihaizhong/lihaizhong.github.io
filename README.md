# 小白的博客仓库

```bash
# 新建博客
npm run create

# 开发环境运行
npm run dev

# 构建生产环境
npm run build

# 发布构建包
pm2 deploy ecosystem.config.js production
```

## serve

```bash
# serve 本地启动
serve -p [port] [dist]
```
