简述 CSS 的盒模型，包括标准盒模型和怪异盒模型的区别，以及如何通过 CSS 进行切换

Jul 11, 2025
查看答案

CSS 盒模型是 CSS 布局的基础，它规定了元素在页面中占据空间的计算方式，每个元素都可以看作一个盒子，由 内容（content）、边框（border）、内边距（padding）、外边距（margin） 四部分组成。

标准盒模型与怪异盒模型的区别

标准盒模型和怪异盒模型（也称为 IE 盒模型）的核心区别在于元素总宽度和总高度的计算方式：

标准盒模型

  • 元素总宽度 = 内容区宽度（width） + 左右内边距（padding-left + padding-right） + 左右边框宽度（border-left + border-right）
  • 元素总高度 = 内容区高度（height） + 上下内边距（padding-top + padding-bottom) + 上下边框宽度（border-top + border-bottom）

也就是说，标准盒模型中设置的 width 和 height 仅指代内容区的尺寸。

怪异盒模型

  • 元素总宽度 = 设置的 width 值（width 包含内容区宽度 + 左右内边距 + 左右边框宽度）
  • 元素总高度 = 设置的 height 值（height 包含内容区高度 + 上下内边距 + 上下边框宽度）

CSS 切换盒模型

通过 CSS 的 box-sizing 属性可以切换这两种盒模型：

  • 当 box-sizing 属性的值为 content-box 时，采用标准盒模型（默认值）。
  • 当 box-sizing 属性的值为 border-box 时，采用怪异盒模型。

例如，若一个元素设置 CSS 属性为 width: 200px; padding: 10px; border: 5px solid #000;，在标准盒模型下，其总宽度为 230px，在怪异盒模型下，其总宽度为 200px。
