如何实现元素的水平居中、垂直居中以及水平垂直居中

Jul 13, 2025
查看答案

水平居中

  • 行内元素/文本
    • 给父元素设置 text-align: center，适用于 <span>、<a>、文本等行内元素。
  • 块级元素（定宽）
    • 给元素设置 margin: 0 auto，需指定 width（否则元素默认占满父容器宽度，居中无效果）。
  • 块级元素（不定宽）
    • Flexbox：父元素设置 display: flex; justify-content: center;，子元素宽度可自适应。
    • Grid：父元素设置 display: grid; justify-content: center;。
  • Position：父元素设置 position: relative;，子元素设置 position: absolute; left: 50%; transform: translateX(-50%);，无需知道子元素宽度。

垂直居中

  • 行内元素/文本(单行)
    • 给父元素设置 line-height 等于父元素高度，适用于单行文本或行内元素。
  • 行内元素/文本(多行)
    • 父元素设置 display: table-cell; vertical-align: middle;，同时指定父元素高度。
  • 块级元素
    • Flexbox：父元素设置 display: flex; aligin-items: center;，兼容性较好且灵活。
    • Grid：父元素设置 display: grid; align-items: center;。
    • Position：父元素设置 position: relative;，子元素设置 position: absolute; top: 50%; left: 50%; transform: translateY(-50%)，无需知道子元素高度。
    • 已知父元素高度且子元素高度固定：子元素设置 margin-top: (父高 - 子高) / 2。

水平垂直居中

  • Flexbox：父元素设置 display: flex; justify-content: center; align-items: center;，最简单常用，支持不定宽高元素。
  • Grid：父元素设置 display: grid; place-items: center;（place-items 是 align-items 和 justify-items 的简写）。
  • Position + transform：父元素设置 position: relative;，子元素设置 position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%)，适用于不定高元素。
  • Position + margin：父元素设置 position: relative;，子元素设置 position: absolute; top: 0; left: 0; right: 0; bottom: 0; margin: auto;，需指定子元素宽高。
  • table-cell：父元素设置 display: table-cell; vertical-align: middle; text-align: center;，子元素设置 display: inline-block;（将会计元素转为行内块，配合父元素文本居中）。
